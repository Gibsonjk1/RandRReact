## Instructions for Build and Use

[Software Demo](https://youtu.be/pGZebZrmOXY)

Steps to build and/or run the software:
To run this project, you will be required to run two separate windows of vs code.

1. Open the RandR-BE file and run "npm i" to install dependencies, then run "npm start" (this will start the back-end on localhost:3000)
2. in a separate window, open the RandR-FE file and run "npm i" to install dependencies, run "npm run build" to build the vite file, then run "npm run preview" to run the file on localhost:4173
3. open localhost:4173 in a browser to see the app.

Instructions for using the software:

1. The first page is just a welcome page, click on the barbell button or navigate to "/exercise" to see the exercise page
2. Currently the exercises are unfiltered and returned in a raw json block.
